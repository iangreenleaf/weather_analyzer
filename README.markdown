# Weather Analyzer #

(For lack of a better name)

## Data import ##

I've been using mostly the [NOAA climate normals](http://www.ncdc.noaa.gov/cdo-web/datasets).

Put em in `data/normals`, and run:

    padrino runner data/normals.rb
